# ExtraMagicWords v1.0.0

This MediaWiki extension provides useful magic words.

## Magic Words

- `{{!!}}` => `||`
- `{{(}}` => `{`
- `{{)}}` => `}`
- `{{((}}` => `{{`
- `{{))}}` => `}}`
- `{{【}}` => `[`
- `{{】}}` => `]`
- `{{【【}}` => `[[`
- `{{】】}}` => `]]`
- `{{.}}` => `·`
- `{{-}}` => `<div class="visualClear"></div>`

- `{{PAGENAMEH}}`
- `{{FULLPAGENAMEH}}`
- `{{ROOTPAGENAMEH}}`
- `{{BASEPAGENAMEH}}`
- `{{SUBPAGENAMEH}}`
- `{{TALKPAGENAMEH}}`
- `{{SUBJECTPAGENAMEH}}`
- `{{CREATEUSER}}`

## Parser Functiions

- `{{PAGENAMEH:}}`
- `{{FULLPAGENAMEH:}}`
- `{{ROOTPAGENAMEH:}}`
- `{{BASEPAGENAMEH:}}`
- `{{SUBPAGENAMEH:}}`
- `{{TALKPAGENAMEH:}}`
- `{{SUBJECTPAGENAMEH:}}`
- `{{CREATEUSER:}}`
